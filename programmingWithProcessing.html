Programmieren lernen - mit Processing

This page is written in German. It was made for a German speaking audience as some kind of tutorial supporting me while live teaching.

Moin Leudde. Powerpoint etc. sind langweilig und doof, ich benutze HTML, CSS und jeglichen anderen Stuff, weil ich Bock drauf hab' und es fuer mich so schneller geht etwas ansehnliches hinzuzimmern, da ich schon ein eigenes Template habe. Ausserdem kann man HTML mit jedem Browser oeffnen und ansehen.
Wenn trotzdem irgendwas kacke aussieht (es ist wirklich nicht schoen...), koennt ihr Euch ja eigene CSS schreiben, das ist das schoene daran.

"Hello world!"

Das arme Kind, das niemand mag

  • ist in den meisten Programmiersprachen nicht mehr als eine Zeile Code
  • macht Dich noch nicht zum Programmierer

println("Hello world!");

Wofuer ist das gut?

Hello world soll das erste und einfachste Beispiel fuer Code sein. Dieser Einzeiler gibt in der Programmkonsole die Zeichenkette "Hello World!" aus.
Ich find's Gruetze, wie die meisten anderen auch. Da wir aber Processing haben, koennen wir den Einstieg etwas anders gestalten.

Processing

Processing bietet uns als frame und IDE eine nette Basis zum Programmieren.
Ganz oben Schaltflaechen zum Starten und Stoppen unseres Programms, mittig ein Scriptfenster und darunter eine Konsolenausgabe - mehr brauchen wir erstmal nicht.
